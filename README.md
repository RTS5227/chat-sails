Install sails beta<br/>
<code>npm -g install sails</code>
Continue:<br>
<code>git clone https://github.com/KnightVincv/chat-sails.git</code><br/>
After:
<code>cd chat-sails</code></br>

<code>npm install sails-mongo</br>
npm install MD5</code><br/>
...
<br/>

<code>
sails lift</code><br>
